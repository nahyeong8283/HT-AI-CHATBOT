# HT 신호등 AI

생산 가능 여부를 자연어로 질문하면 원료, 설비, 인력, 납기 조건을 확인하고 결과를 신호등처럼 보여주는 AI Agent 프로토타입입니다.

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

## GitHub Pages 배포 방법

1. GitHub에서 새 Repository를 만듭니다.
2. 이 폴더의 `index.html`과 `README.md`를 업로드합니다.
3. Repository의 `Settings`로 이동합니다.
4. `Pages` 메뉴를 엽니다.
5. `Deploy from a branch`를 선택합니다.
6. Branch는 `main`, 폴더는 `/root`를 선택합니다.
7. 저장하면 몇 분 뒤 GitHub Pages 주소가 생성됩니다.

## 주요 기능

- 자연어 생산 질문 입력
- 생산 가능/불가 판단
- 원료, 설비, 인력, 납기 게이지 표시
- AI 추천 문구 표시
- 샘플 ERP/MES 데이터 기반 시뮬레이션
